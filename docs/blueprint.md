# **App Name**: CampusFind

## Core Features:

- User Authentication: Allow users to register and log in securely using Firebase Authentication, supporting email/password login.
- Item Reporting: Enable users to submit reports for lost or found items via a structured form, including item name, description, location, category, date, and optional image upload.
- Item Display and Filtering: Display reported items in a filterable list, showing the most recent items first and allowing filtering by lost/found status, category, or item name search.
- Data Storage: Store lost and found item reports in separate Firestore collections, organizing data for easy retrieval and management.
- User Dashboard: Allow users to view their own submitted reports in a dedicated dashboard.
- Report Deletion: Enable users to delete their own reports from the system.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to create a trustworthy and reliable feel.
- Background color: A very light blue (#E3F2FD), close to white, to provide a clean and open interface.
- Accent color: A contrasting orange (#FF9800) to highlight important actions and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif, for clear readability and a modern yet approachable feel.
- Use simple, clear icons from a library like FontAwesome to represent item categories and actions.
- Maintain a clean and organized layout with consistent spacing and clear visual hierarchy to ensure ease of use and navigation.
- Employ subtle transitions and animations for user interactions to enhance engagement and provide feedback.
